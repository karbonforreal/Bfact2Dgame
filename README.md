# Bfact2Dgame

Browser-based Cyberdog-inspired top-down shooter prototype.

## Run locally

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080` in Safari, Edge, Chrome, or Firefox.

## Controls

- `WASD` move
- Mouse aim
- Left click fire
- `R` reload
- `M` switch maps (debug/testing)

## Beta 1 includes

- Two hand-built maps
- Camera centered on player with world movement
- Enemy AI with projectile combat
- Bullet visuals and hit tracking for both sides
- Ammo, health, wins/losses, and kill counters
